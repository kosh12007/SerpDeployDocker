# System Patterns: SERP Анализатор

## Архитектура

Проект построен на базе Flask с модульной структурой (Blueprints), что обеспечивает разделение ответственности и легкость поддержки.

## Ключевые паттерны

- **Active Record (Models)**: Все модели данных в `app/models.py` реализуют логику работы с БД внутри себя (`create`, `get`, `update`, `delete`).
- **Blueprints**: Функциональность разделена на модули (auth, projects, parsing, ai, api и др.).
- **Application Factory**: Инициализация приложения в `app/__init__.py`.
- **Multithreading**: Длительные операции (парсинг позиций, AI-анализ) выполняются в отдельных потоках (`threading.Thread`) для предотвращения блокировки основного процесса веб-сервера. Состояние задач отслеживается через таблицы сессий.

## Структура данных

- **MySQL**: Основное хранилище.
- **Active Record**: Нативный SQL и параметризованные запросы в моделях.
- **Справочники**: География (Яндекс Регионы, Google Locations), Устройства, Типы поиска.

## Важные технические решения

- **XmlRiver Integration**: Основной источник данных SERP.
- **Custom CSS Framework**: Использование Tailwind CSS для быстрой разработки UI.
- **Vanilla JS**: Минималистичный фронтенд без тяжелых фреймворков, использование `fetch API` для взаимодействия с бэкендом и динамического обновления таблиц позиций.
- **Limit Estimation**: Предварительный расчет стоимости парсинга перед запуском задачи для контроля баланса пользователя.
- **CSRF Protection**: Глобальная защита от CSRF-атак с использованием Flask-WTF. Все изменяющие запросы должны содержать валидный токен.
- **Logging**: Детальное логирование каждого модуля в папку `logs/`.

## Безопасность и CSRF

- **Формы**: Каждая HTML-форма должна содержать скрытое поле с токеном:
  ```html
  <input type="hidden" name="csrf_token" value="{{ csrf_token() }}" />
  ```
- **AJAX запросы**: Для всех POST, PUT, DELETE запросов через `fetch` необходимо добавлять заголовок `X-CSRFToken`:
  ```javascript
  headers: {
      'Content-Type': 'application/json',
      'X-CSRFToken': document.querySelector('meta[name="csrf-token"]').getAttribute('content')
  }
  ```
- **Мета-тег**: Токен JS получает из мета-тега в `layout.html`:
  ```html
  <meta name="csrf-token" content="{{ csrf_token() }}" />
  ```
