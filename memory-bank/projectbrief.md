# Project Brief: SERP (Search Engine Results Page) Анализатор

## Цели проекта

Создание мощного и гибкого инструмента для мониторинга и анализа поисковой выдачи (SERP) в Google и Яндекс.

## Основные возможности

- **Парсинг позиций**: Автоматизированный сбор данных о позициях сайтов по ключевым запросам.
- **Анализ конкурентов**: Выгрузка и анализ ТОП-10 результатов поиска.
- **Кластеризация**: Группировка запросов на основе семантической близости и схожести выдачи.
- **AI Интеграция**: Использование нейросетей (DeepSeek) для анализа контента и генерации текстов (статьи, ТЗ).
- **Технический аудит**: Проверка HTTP-статусов, редиректов и системный анализ страниц.
- **Текстовый анализатор**: Глубокий анализ контента (LSI, n-граммы, плотность слов, тошнота).
- **Проверка уникальности**: Многопоточная проверка текстов на плагиат с использованием Qdrant и кеширования.
- **Частотность запросов**: Сбор данных о популярности ключевых фраз из Яндекс.Wordstat.

## Ключевые метрики успеха

- Точность и актуальность данных парсинга.
- Высокая производительность при обработке больших объемов запросов.
- Удобный и современный пользовательский интерфейс.
- Масштабируемость системы для добавления новых поисковых систем и инструментов.
