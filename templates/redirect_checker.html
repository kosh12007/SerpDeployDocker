Проверка редиректов

{% set info_content %}

Этот инструмент позволяет массово проверить редиректы для разных вариаций URL, чтобы убедиться в правильной настройке сервера.

  1. Генерация списка: Введите URL внутренней страницы сайта (например, `https://example.com/category/page`), чтобы сгенерировать список тестовых URL. Это позволит проверить редиректы для www/без-www, http/https, с/без слеша, а также чувствительность к регистру.
  2. Редактирование: Вы можете вручную отредактировать сгенерированный список URL перед запуском проверки.
  3. Проверка: Инструмент отправит запросы ко всем URL из списка и покажет итоговый код ответа, цепочку редиректов и конечный URL.
