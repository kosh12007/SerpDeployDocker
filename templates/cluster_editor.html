Инструмент Hard-кластеризации

Загрузит ключевые слова из выбранного проекта.

Как это работает?

  1. Выберите проект из списка.
  2. Убедитесь, что для ключевых слов в проекте хотя бы раз снимались позиции (это необходимо для получения данных SERP).
  3. Если позиции не снимались, перейдите в проект, снимите позиции и вернитесь на эту страницу.
  4. Выберите "порог совпадений" — минимальное количество общих URL для объединения запросов в одну группу.
  5. Нажмите "Выполнить кластеризацию" для предпросмотра.
