## Технологический стек

### Backend

- **Python** 3.x — основной язык программирования
- **Flask** — веб-фреймворк
  - **Flask-Login** — управление аутентификацией пользователей
  - **Flask-Mail** — отправка email (восстановление пароля)
  - **Flask-Caching** — кеширование данных (SimpleCache)
- **MySQL** — основная база данных
  - **mysql-connector-python** — драйвер для работы с MySQL
- **Werkzeug** — хеширование паролей и безопасность

### Frontend

- **HTML5** — разметка страниц
- **Tailwind CSS** — utility-first стилизация
  - **PostCSS** — препроцессор для CSS
  - **Autoprefixer** — автоматические вендорные префиксы
- **Vanilla JavaScript** — клиентская логика без фреймворков
- **Flatpickr** — библиотека для работы с датами

### Работа с данными

- **pandas** — анализ и обработка данных
- **openpyxl** — работа с Excel файлами (экспорт/импорт)
- **beautifulsoup4** — парсинг и анализ HTML

### AI и машинное обучение

- **openai** — интеграция с DeepSeek API через OpenRouter
- **scikit-learn** — алгоритмы машинного обучения
- **nltk** — обработка естественного языка (NLP)

### Внешние API

- **XmlRiver API** — парсинг поисковой выдачи Google и Yandex
- **DeepSeek API** — генерация контента и анализ текста

### Разработка и тестирование

- **pytest** — фреймворк для тестирования
- **python-dotenv** — управление переменными окружения

### Развертывание

- **Passenger WSGI** — развертывание на хостинге (passenger_wsgi.py)
- **Node.js** — для сборки Tailwind CSS (npm)
