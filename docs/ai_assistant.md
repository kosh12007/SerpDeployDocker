## AI Ассистент

Этот модуль предоставляет интерфейс для взаимодействия с нейросетью DeepSeek через API OpenRouter.

### Страница "AI Ассистент" (`/ai`)

-   **Назначение:** Отображает страницу с формой для отправки запросов к AI.
-   **Метод:** `GET`
-   **Доступ:** Требуется аутентификация.

На странице доступны два поля:
-   **Системный промпт:** Позволяет задать контекст или роль для AI. По умолчанию: "Ты — полезный ассистент.".
-   **Ваше сообщение:** Основное текстовое поле для ввода запроса пользователя.

### API эндпоинт `/ask-ai`

-   **Назначение:** Обрабатывает AJAX-запрос со страницы "AI Ассистент", отправляет данные в нейросеть и возвращает ответ.
-   **Метод:** `POST`
-   **Доступ:** Требуется аутентификация.
-   **Параметры (JSON):**
    -   `system_prompt` (string): Текст системного промпта.
    -   `user_message` (string): Сообщение пользователя.
-   **Ответ (JSON):**
    -   В случае успеха: `{"response": "Текст ответа от AI"}`
    -   В случае ошибки: `{"error": "Описание ошибки"}`
