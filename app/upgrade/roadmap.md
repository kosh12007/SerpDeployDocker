# План реализации нового функционала

## Фаза 1: Базовая структура
- [ ] **Модели БД (SQLAlchemy):**
  - Создать/обновить модели для таблиц: `projects`, `parsing_variants`, `queries`, `parsing_position_results`, `query_groups` и всех связанных справочников (`search_engines`, `search_types`, `yandex_regions`, `locations`, `devices`).
- [ ] **CRUD для проектов:**
  - Реализовать эндпоинты `GET`, `POST`, `PUT`, `DELETE` для `/api/projects`.
- [ ] **CRUD для вариантов парсинга:**
  - Реализовать эндпоинты `GET`, `POST`, `DELETE` для `/api/projects/<project_id>/variants`.
- [ ] **Управление запросами:**
  - Реализовать эндпоинты для добавления (`POST /api/projects/<project_id>/queries`), загрузки CSV (`POST /api/projects/<project_id>/queries/upload`) и удаления (`DELETE /api/queries/<id>`) запросов.
