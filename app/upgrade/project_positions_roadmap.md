# План реализации отложенного парсинга позиций

Этот документ отслеживает прогресс реализации асинхронного парсинга с подтверждением через модальное окно и учетом лимитов.

- [x] **Анализ текущей реализации:** Анализ `project_positions.html` и связанных эндпоинтов для понимания текущей синхронной логики.
- [x] **Разработка схемы модального окна:** Проектирование HTML-структуры для модального окна подтверждения.
- [x] **Реализация JavaScript-логики для прогноза:** Создание JS-кода для запроса оценки лимитов и отображения модального окна.
- [x] **Flask-эндпоинт для прогноза:** Реализация эндпоинта для расчета и возврата предполагаемых затрат лимитов.
- [x] **Flask-эндпоинт для запуска задачи:** Модификация эндпоинта для создания асинхронной задачи и возврата ее ID.
- [x] **Эндпоинт для статуса задачи:** Создание эндпоинта для получения статуса выполнения задачи по ее ID.
- [x] **JavaScript-логика для опроса статуса:** Реализация JS-кода для периодического запроса статуса задачи и обновления UI.
- [x] **Интеграция списания лимитов:** Финальная проверка и интеграция механизма списания лимитов после выполнения задачи.
